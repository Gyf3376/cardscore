# 图标资源说明

本目录存放小程序使用的图标资源。

图标使用 Lucide Icons 风格的 SVG 格式，为了保证小程序正常运行，请确保以下图标文件存在：

- `message-circle.svg` - 消息/聊天图标（登录页）
- `shield-check.svg` - 安全/授权图标（登录页）
- `users.svg` - 用户群组图标（房间页、等待大厅）
- `plus.svg` - 加号图标（创建房间、添加玩家）
- `log-in.svg` - 登录图标（加入房间）
- `chevron-right.svg` - 右箭头图标（导航）
- `x.svg` - 关闭/取消图标
- `settings.svg` - 设置图标（房间设置）
- `qrcode.svg` - 二维码图标（扫码加入）
- `info.svg` - 信息图标（规则说明）
- `rotate-ccw.svg` - 重置/撤销图标（重置游戏）
- `award.svg` - 奖杯图标（积分榜）
- `trending-up.svg` - 上升趋势图标（赢）
- `trending-down.svg` - 下降趋势图标（输）
- `sparkles.svg` - 闪光图标（空状态）
- `plus-circle.svg` - 加号圆形图标（记录得分）
- `history.svg` - 历史记录图标（战报）
- `book-open.svg` - 打开书本图标（规则说明）
- `layers.svg` - 图层图标（基本规则）
- `zap.svg` - 闪电图标（炸弹）
- `alert-circle.svg` - 警告圆圈图标（结算逻辑）
- `trash-2.svg` - 删除图标（删除记录）

这些图标可以从 [Lucide Icons](https://lucide.dev/) 官网下载。
