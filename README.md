# 扑克计分专家 - 微信小程序版

这是一个基于微信小程序的AI记账助手，使用Gemini API提供智能记账服务。

## 开发环境要求

- 微信开发者工具
- 微信小程序账号

## 运行方式

1. 在微信开发者工具中打开项目
2. 在"开发环境配置"中设置`GEMINI_API_KEY`为您的Gemini API密钥
3. 点击"编译"按钮运行项目

## 配置API密钥

在小程序启动时需要配置Gemini API密钥：
1. 在小程序首页进入设置页面
2. 输入您的Gemini API密钥
3. 保存后即可正常使用AI功能
